# Changelog

## v1.0.0 - 2026-03-31

AI-Meeting-Framework の初期安定版。

### 含まれるもの

- 複数 AI と人間が会議を行うための基本構造
- `Council Mode`
- `Live Council Mode`
- `character` 表示
- `teaching` 表示
- `yang-council` による教える形式の会議
- `summary`、`report`、`transcript` の三層出力
- bootstrap と meeting type ルーティング
- 日本語ベースの主要テンプレート
- teaching 用ガイドと記事種別テンプレート
- encoding ルールと UTF-8 確認スクリプト
- 実サンプルと保存済み会議セッション群

### 位置づけ

このバージョンは、会議フレームワークとしての初期安定版ベースラインとする。  
今後の変更は、大きな作り直しではなく、改善提案をもとに小さく積み上げる。
