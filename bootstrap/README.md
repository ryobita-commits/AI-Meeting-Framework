# bootstrap

このフォルダは、会議を始める前に最低限の前提を揃えるための入口です。

## 基本フロー

1. `meeting-type-router.md` で会議種別を決める
2. `questions/` の対応ファイルで初期質問に答える
3. 回答を `initial-answers.md` に記録する
4. `session-bootstrap_template.md` に沿って agenda や会議設定を作る
5. 会議を実施し、`sessions/` に成果物を残す

## 会議種別

- `casual`
- `planning`
- `review`
- `decision`
