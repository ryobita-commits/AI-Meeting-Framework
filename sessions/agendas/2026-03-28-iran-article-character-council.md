# agenda

## 会議名
イラン指導部と対米秘密交渉に関する記事の信ぴょう性評価

## 日付
2026-03-28

## モード
Human-in-the-Loop / Council Mode / character

## テーマ
イランの最高指導層の崩壊、ガリバフ議長の水面下交渉、モジタバ師の求心力低下などを扱う記事について、事実、推測、誇張を分けて信ぴょう性を評価する

## 参加者が知りたいこと
この記事がどこまで信じられるのか、どこが未確認情報に依存しているのか、面白いが危うい部分はどこか

## 会議の目的
- 事実として比較的強い部分を分ける
- 推測や物語化が強い部分を分ける
- 反論と再反論を整理する
- 信ぴょう性を 5 段階で評価する

## 議論したい主な点
- ハメネイ師死亡、ラリジャニ殺害などの前提事実の確度
- ガリバフ議長が実質的な交渉相手だという分析の妥当性
- モジタバ師の消息や求心力低下に関する未確認情報の扱い
- 強硬派ジャリリ氏やタンシリ司令官の反発リスクの描き方

## 使う役割
- moderator: main assistant
- members: analyst, advocate, critic
- human: user

## 会議終了時に欲しい成果物
- transcript
- meeting note
- decision memo
- action list
- summary
- report
